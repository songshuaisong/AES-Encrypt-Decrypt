# linux
If it is a Linux, it depends on the following environment:

    - Ubuntu16.04 64Bits LTS
    - cmake version 3.5.1 (Necessary!)
    - gcc version 5.4.0 (Necessary!)
    
    And use the following commands : 
        - cd build.unix
        - cmake -G"Unix Makefiles" ../
        - make
        - ./sm4test
